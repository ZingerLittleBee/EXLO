**EXLO** is a self-hosted SSH reverse tunnel service designed for private teams and Homelabs. No client installation required - use the standard `ssh -R` command to expose your local services to the public internet.

## Quick Overview

```bash
# Connect to your EXLO server
ssh -R 8000:localhost:8000 -p 2222 user@your.domain

# Open the authentication link in your browser for Device Flow
# After authentication, visit: https://your-subdomain.your.domain
```
