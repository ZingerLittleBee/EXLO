EXLO provides a Management API for tunnel management.

## Available APIs

| API | Port | Description | Authentication |
|-----|------|-------------|----------------|
| [Management API](/en/api-reference/management-api) | `:9090` | Tunnel management | None (internal access only) |

> [!CAUTION]
> The Management API runs on port `:9090` and requires **no authentication**. Ensure this port is not exposed to the public internet and only accessible by the Web Dashboard container.

## Common Response Format

All APIs return JSON format:

### Success Response

```json
{
  "success": true,
  "message": "Operation completed"
}
```

### Error Response

```json
{
  "error": "Error description"
}
```
