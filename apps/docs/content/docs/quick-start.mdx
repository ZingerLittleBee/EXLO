## Prerequisites

- Docker & Docker Compose
- A public domain (e.g., `your.domain`)
- Wildcard DNS or subdomain configuration support

## One-Click Start

```bash
# Clone the repository
git clone https://github.com/ZingerLittleBee/EXLO
cd EXLO

# Copy environment variable template
cp .env.example .env

# Edit configuration
# Set DATABASE_URL, BASE_DOMAIN, etc.

# Start services
docker compose up -d
```

## First Connection

### 1. Create SSH Tunnel

```bash
ssh -N -R 80:localhost:3000 -p 2222 user@your.domain
```

Parameter explanation:
- `-N`: Don't execute remote commands
- `-R 80:localhost:3000`: Forward remote port 80 to local port 3000
- `-p 2222`: SSH server port


## Disconnect

In SSH reverse tunnel mode, `Ctrl+C` won't terminate the connection. Use the following escape sequence:

```
Press Enter, then type ~.
```
